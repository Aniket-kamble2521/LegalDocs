# Security Policy

We take the security of LegalDocs very seriously. If you believe you have found a security vulnerability in this application, please report it to us immediately through the proper channels.

## Supported Versions

Only the latest release (currently v1.0.0) is actively supported with security updates.

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please report security issues by emailing: **security@legaldocs.co**

Please include the following details in your report:
- Type of issue (e.g. CSRF, path traversal, IDOR, SQL injection)
- Step-by-step instructions to reproduce the issue
- Potential impact
- Any suggested remediations

We will review your submission and respond within 48 hours to coordinate a security patch.
